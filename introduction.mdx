## What is Wiki.js?

Wiki.js is a self-hosted knowledge base platform designed for teams who want full control over their documentation. It stores content in a database (PostgreSQL, MySQL, MariaDB, MS SQL Server, or SQLite) and can optionally sync to Git repositories, cloud storage, or other backends.

Key capabilities include:

- **Multiple editors** — Choose from Markdown, WYSIWYG (CKEditor), AsciiDoc, raw HTML, or the REST/GraphQL API
- **Authentication flexibility** — 21+ providers including Local, LDAP, OAuth2, SAML, Google, GitHub, and more
- **Storage backends** — Git, AWS S3, Azure Blob, Google Drive, Dropbox, SFTP, and more
- **Full-text search** — PostgreSQL built-in, Elasticsearch, Algolia, Solr, and others
- **Access control** — Group-based permissions with page-level rules
- **GraphQL API** — Full programmatic access to all content and admin operations
